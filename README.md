# My Personal Portfolio

This is a modern React website built with Vite. It is designed to be a personal portfolio.

## Important Note

**Node.js was not found in your current environment.**
To run this project, you must have Node.js installed.

## How to Run

1.  **Install Node.js**: Download and install from [nodejs.org](https://nodejs.org/).
2.  **Open Terminal**: Navigate to this directory.
3.  **Install Dependencies**: Run the following command:
    ```bash
    npm install
    ```
4.  **Start Development Server**: Run:
    ```bash
    npm run dev
    ```
5.  **View Website**: Open the URL shown in the terminal (usually `http://localhost:5173`).

## Project Structure

- `src/App.jsx`: The main component containing the website layout (Hero, About, Projects, Contact).
- `src/index.css`: Styling for the website.
- `src/main.jsx`: Entry point for React.

## Customization

- Edit `src/App.jsx` to change the text and sections.
- Edit `src/index.css` to change colors and styles.
